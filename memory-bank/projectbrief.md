# Project Brief

## Core Requirements
- Build a policy analysis web application using Next.js 14 with App Router
- Implement Prisma ORM for database operations with PostgreSQL
- Create responsive UI for policy document visualization
- Support user authentication and role-based access control

## Goals
- Provide intuitive interface for parsing and analyzing complex policy documents
- Achieve 95%+ test coverage for critical components
- Maintain clean architecture with separation of concerns
- Ensure mobile-responsive design across all views

## Scope
- Frontend: Next.js application with TypeScript
- Backend: Prisma data layer with PostgreSQL database
- Features: Document upload, analysis dashboard, results visualization
- Out of scope: Mobile app development, third-party API integrations beyond auth
